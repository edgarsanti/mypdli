Eliminación de acentos¶

A continuacin, se define la funcin remove_accents, la cual tiene como propsito eliminar los acentos del texto.
Para ello, primero se normaliza la cadena mediante la forma NFKD de unicodedata, lo que descompone los caracteres acentuados en su letra base y el acento por separado. Posteriormente, se construye una nueva cadena uniendo nicamente aquellos caracteres que no corresponden a marcas de los acentos, verificado a travs de unicodedata.combining(c). El resultado se almacena en la variable text_no_accents, que contiene el texto limpio de acentos y listo para el procedimiento posterior.

#texto sin acentos
import unicodedata
def remove_accents(input_str):
    nfkd_form = unicodedata.normalize('NFKD', input_str)
    return u"".join([c for c in nfkd_form if not unicodedata.combining(c)])
text_no_accents = remove_accents(text_lower)

# Remueve caracteres especiales
def remove_special_chars(input_str):
  chars_to_remove = ";:,.\\-\"'/()[]?!{}~<>|-"
  cleaned_text = input_str
  for char in chars_to_remove:
    cleaned_text = cleaned_text.replace(char, "")
  return cleaned_text

Conteo de la frecuencia de cada palabra en el conjunto de textos¶

En esta seccin se define la funcin word_freq, cuya finalidad es calcular la frecuencia de aparicin de cada palabra en el texto.
La funcin separa el contenido en una lista de palabras y, mediante un ciclo for, va construyendo un diccionario (freq) donde las claves corresponden a las palabras y los valores representan el nmero de veces que aparecen.

Una vez calculadas las frecuencias con el texto limpio (cleaned_text), los resultados se almacenan en la variable freq_words.
Posteriormente, la lista se ordena de mayor a menor frecuencia y se muestran las 35 palabras ms frecuentes como vista preliminar de los resultados.

In [ ]:
# Frecuencia de palabras
def word_freq(input_str):
    freq = {}
    words = input_str.split()   # separar por espacios en lista de palabras
    for word in words:
        if word in freq:
            freq[word] += 1
        else:
            freq[word] = 1
    return freq

# Calcular
freq_words = word_freq(cleaned_text)

# Ordenar
freq_sorted_words = sorted(freq_words.items(), key=lambda x: x[1], reverse=True)

# Mostrar las 35 con mayor frecuencia

Generación de un histograma con las palabras más frecuentes, ordenadas de mayor a menor¶

Para visualizar los resultados comentados en el texto anterior, se define la funcin plot_word_histogram, que permite graficar un histograma con las palabras ms frecuentes del texto.
La funcin recibe como entrada la lista de frecuencias ordenadas y un parmetro (most_freq) que indica cuntas palabras se desean mostrar.

El siguiente grfico muestra que en el libro 1 predominan palabras funcionales propias del espaol como que, de, y, la, no, a y el, las cuales son esenciales para dar estructura a a los textos y explican sus altas frecuencias. Entre las 35 palabras ms frecuentes aparecen los nombres propios como Augusto y Eugenia, esto por la presencia de interacciones entre personajes.

In [ ]:
import matplotlib.pyplot as plt
import numpy as np

def plot_word_histogram(freq_sorted_words, most_freq=35):
    # Limitar
    data = freq_sorted_words[:most_freq]

    # Separar palabras y frecuencias
    words = [item[0] for item in data]
    counts = [item[1] for item in data]

    # Crear
    plt.figure(figsize=(12, 6))
    bars = plt.bar(words, counts, color='#00BFBF')

    # Etiquetas
    for bar in bars:
        yval = bar.get_height()
        plt.text(bar.get_x() + bar.get_width()/2, yval + 0.5, int(yval),
                 ha='center', va='bottom', fontsize=8)

    # Titulos
    plt.xlabel("Palabras")
    plt.ylabel("Frecuencia")
    plt.title(f"Palabras con mayor frecuentes")
    plt.xticks(rotation=90, ha='right')
    plt.tight_layout()
    plt.show()

#  Graficar las 35 palabras ms frecuentes
plot_word_histogram(freq_sorted_words, most_freq=35)

def count_words(input_str):
    words = input_str.split()   # separa por espacios
    return len(words)

def count_unique_words(text: str) -> int:
    words = text.split()
    unique_words = set(words)   # elimina duplicados
    return len(unique_words)
def count_hapax(text: str) -> int:
    freq = word_freq(text)   # diccionario palabra  frecuencia
    hapax = [w for w, c in freq.items() if c == 1]
    return len(hapax)
