import unicodedata
def remove_accents(input_str):
    nfkd_form = unicodedata.normalize('NFKD', input_str)
    return u"".join([c for c in nfkd_form if not unicodedata.combining(c)])

# Remueve caracteres especiales
def remove_special_chars(input_str):
  chars_to_remove = ";:,.\\-\"'/()[]?!{}~<>|"
  cleaned_text = input_str
  for char in chars_to_remove:
    cleaned_text = cleaned_text.replace(char, "")
  return cleaned_text

Para calcular la frecuencia de cada carcter en el texto, se define la funcin char_freq. Esta funcin inicializa un diccionario vaco (freq) y luego recorre cada carcter del texto limpio (cleaned_text). Por cada carcter, verifica si ya existe en el diccionario; si es as, incrementa su contador en uno; de lo contrario, lo aade con un valor inicial de 1. Finalmente, la funcin devuelve el diccionario con las frecuencias. Despus de obtener las frecuencias se ordena de mayor a menor usando sorted() y una funcin lambda para especificar que se ordene por el valor de la frecuencia (x[1]).

El resultado del anlisis de frecuencia de los caracteres del texto, mostrando los 10 con ms frecuencia. Se obseerva que la letra 'a' fue el carcter con mayor frecuencia, seguida por el espacio en blanco, que es fundamental para separar las palabras. Le siguen letras como 'l', 'r' y 's', que son muy comunes.

In [ ]:
# Frecuencia de caracteres
def char_freq(input_str):
  freq = {}
  for char in input_str:
    if char in freq:
      freq[char] += 1
    else:
      freq[char] = 1
  return freq

freq = char_freq(cleaned_text)

# Ordenar por mayor a menos frecuencia
freq_sorted = sorted(freq.items(), key=lambda x: x[1], reverse=True)

Para visualizar la frecuencia de los caracteres, utilizamos la biblioteca matplotlib.pyplot para crear un histograma. La funcin plot_histogram_from_sorted toma la lista ordenada de frecuencias y la utiliza para crear un grfico de barras. Dentro de la funcin, se extraen los caracteres y sus respectivos conteos para ser representados en los ejes X e Y. Se aaden etiquetas a los ejes, un ttulo al grfico, entre otros complementos.

La grfica permite ver de forma clara y rpida la distribucin de las frecuencias. Cada barra del histograma representa un carcter, y su altura muestra cuntas veces apareci en el texto, lo que ayuda a identificar rpidamente cules son los elementos ms y menos frecuentes.

In [ ]:
import matplotlib.pyplot as plt

def plot_histogram_from_sorted(freq_sorted):
  # Extrae los caracteres y las frecuencias de la lista ordenada
  chars = [item[0] for item in freq_sorted]
  counts = [item[1] for item in freq_sorted]

  # Crea el histograma
  plt.figure(figsize=(12, 6))
  bars = plt.bar(chars, counts, color='#00BFBF')

  # Agrega etiquetas de datos a cada barra
  for bar in bars:
    yval = bar.get_height()
    plt.text(bar.get_x() + bar.get_width()/2.0, yval + 0.5, int(yval), va='bottom', ha='center', fontsize=8)

  plt.xlabel("Caracteres")
  plt.ylabel("Frecuencia")
  plt.title("Histograma de Frecuencia de Caracteres")
  plt.xticks(rotation=45, ha='right')
  plt.tight_layout()
  plt.show()
plot_histogram_from_sorted(freq_sorted)
